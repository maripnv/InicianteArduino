## O ambiente de desenvolvimento (IDE)
A Arduino IDE é um software de código aberto (open source), operado em funções C e C++. 
A IDE pode ser instalada gratuitamente clicando [aqui](https://www.arduino.cc/en/software), caso apareça alguma mensagem referindo a doação, saiba que não é preciso doar para baixá-la.
